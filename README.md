# Contato — ENAJU

Páginas de contato para onde o QR do verso dos [cartões de visita](https://github.com/cairesmachado-svg/ENAJU) aponta, em vez de carregar o vCard direto. Cada página oferece "Salvar contato" (baixa o `.vcf`), WhatsApp, e-mail institucional e o site da ENAJU.

**Não editar os arquivos aqui à mão.** Todo o conteúdo é gerado por
`gerar_paginas_contato.py`, no repositório principal (`00_CENTRAL_ENAJU/Gestão
ENAJU_CODE/Atividades_CODE/Cartões de visita/`), a partir de `pessoas.yml`.
Para atualizar um contato, edite o YAML lá, rode o script e faça commit aqui.

Publicado via GitHub Pages (branch `main`, raiz `/`).
